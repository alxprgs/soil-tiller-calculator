# Soil Tiller Calculator

**Версия:** 0.2.4

Приложение для автоматизированного расчёта параметров почвообрабатывающих лап.
Программа помогает рассчитать силу тяги, удельное сопротивление, тяговую
мощность и ориентировочный расход топлива для серийной лапы КПС-4,0,
экспериментальной лапы и пользовательских инструментов.

## Возможности

- расчёт силы тяги `F`, Н;
- расчёт удельного сопротивления `q`, Н/м;
- расчёт тяговой мощности `P`, кВт;
- расчёт расхода топлива `Q`, л/ч на одну лапу;
- сравнение двух инструментов по удельному сопротивлению;
- ручной ввод скорости или автоматический поиск оптимальной скорости;
- пользовательские пороги скорости и глубины обработки;
- настройка шага перебора скоростей для оптимизации;
- создание, дублирование и удаление пользовательских инструментов;
- импорт и экспорт JSON-конфигов;
- выбор постоянного места хранения конфига;
- русская и английская локализация;
- адаптивный интерфейс и графики.

## Установка

### Windows через Chocolatey

Установите приложение из Chocolatey Community Repository:

```powershell
choco install soil-tiller-calculator
```

После установки приложение доступно из любой директории:

```powershell
soil-tiller-calculator
stc
```

### Ubuntu через APT-репозиторий

Подключите ключ репозитория:

```bash
sudo apt update
sudo apt install curl gnupg

curl -fsSL https://alxprgs.github.io/soil-tiller-calculator/soil-tiller-calculator-archive-keyring.asc \
  | sudo gpg --dearmor -o /usr/share/keyrings/soil-tiller-calculator-archive-keyring.gpg
```

Добавьте APT-репозиторий:

```bash
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/soil-tiller-calculator-archive-keyring.gpg] https://alxprgs.github.io/soil-tiller-calculator stable main" \
  | sudo tee /etc/apt/sources.list.d/soil-tiller-calculator.list
```

Установите приложение:

```bash
sudo apt update
sudo apt install soil-tiller-calculator
```

Если `apt` сообщает `Unable to locate package soil-tiller-calculator`, обновите
индексы репозиториев принудительно:

```bash
sudo apt clean
sudo rm -rf /var/lib/apt/lists/*alxprgs.github.io*
sudo apt update
sudo apt install soil-tiller-calculator
```

Если в системе не работает IPv6, можно заставить `apt` использовать IPv4:

```bash
sudo tee /etc/apt/apt.conf.d/99force-ipv4 >/dev/null <<'EOF'
Acquire::ForceIPv4 "true";
EOF
sudo apt update
```

После установки приложение доступно из любой директории:

```bash
soil-tiller-calculator
```

Для запуска графического интерфейса нужен рабочий дисплей. На обычной Ubuntu
Desktop приложение откроется как обычное окно; в WSL может понадобиться WSLg
или другой X-сервер.

### Из исходников

Требуется Python 3.11 или новее.

```powershell
python -m pip install -e .[test]
```

На Linux может понадобиться системный пакет Tkinter, например:

```bash
sudo apt install python3-tk
```

## Запуск

После установки приложение можно запустить командой:

```powershell
soil-tiller-calculator
```

Для установок через Chocolatey и APT также доступна короткая команда:

```powershell
stc
```

Также можно запустить модуль напрямую:

```powershell
python -m soil_tiller_calculator
```

## Как пользоваться

1. Укажите глубину обработки.
2. Выберите режим инструмента: один инструмент, сравнение КПС/экспериментальной
   лапы или сравнение выбранных инструментов.
3. Выберите режим скорости: ручной ввод или автоматический поиск.
4. Нажмите `Выполнить расчёт`.

Результаты выводятся справа. В текстовом блоке есть расшифровка обозначений:

- `F` — сила тяги;
- `q` — удельное сопротивление;
- `P` — тяговая мощность;
- `Q` — расход топлива.

## Настройки

Основные настройки находятся в верхнем меню:

- `Файл` — импорт/экспорт JSON, выбор места хранения конфига, возврат к
  стандартному месту хранения;
- `Инструменты` — управление рабочими инструментами;
- `Настройки` — язык интерфейса, пользовательские пороги скорости и глубины,
  шаг оптимизации.

По умолчанию конфиг хранится:

- Windows: `%APPDATA%/SoilTillerCalculator/config.json`;
- Linux: `$XDG_CONFIG_HOME/soil-tiller-calculator/config.json` или
  `~/.config/soil-tiller-calculator/config.json`.

Если пользователь выбирает другое место хранения, приложение запоминает этот
путь через служебный файл `config-location.json`.

## Пользовательские инструменты

Инструмент задаётся набором параметров:

- `id` — технический идентификатор;
- название;
- ширина захвата, м;
- опорная глубина, см;
- опорные точки `скорость:сила`;
- цвет и стиль линии на графике.

Для расчёта силы между опорными точками используется линейная интерполяция.
За пределами диапазона опорных точек используется линейная экстраполяция.
